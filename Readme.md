# 🕵️‍♂️ fStat:

 Get All Kinds of Stats from a Directory and Its Subdirectories

## 📦 Install :

### from pnpm :
  - `pnpm i -g fstat`
  - `pnpx fstat --help`

### from source :
  - `git clone https://github.com/Benoit-Welsch/fStat.git`
  - `cd fStat`
  - `bun install`
  - `bun run -- --help`
